# LearningSpace

## Clone this repository and navigate to folder
git clone https://github.com/starreslzx/LearningSpace.git
cd LearningSpace
## Create environment
conda create --name LearningSpace python=9.12

conda activate LearningSpace
## Install Package
pip install -r requirements.txt
## Quick start
python code/main.py
## Note
If you want to experience extracting question from picture,
you need to download pytesseract and change the path accordingly

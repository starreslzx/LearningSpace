LearningSpace - 智能学习助手
📖 项目概述
LearningSpace 是一款专为学生和自学者设计的综合性学习管理桌面应用。它结合了任务管理、专注训练、AI智能辅导和题库练习等多种功能，帮助用户建立高效的学习习惯。

🎯 核心功能
📝 待办事项管理
任务创建与管理：轻松添加、编辑和删除学习任务

智能分类：支持优先级标记和任务分类

进度追踪：实时查看任务完成状态

批量操作：一键清理已完成任务

⏱️ 专注模式（番茄钟）
可定制时长：支持5分钟到2小时的专注时段

防误操作：三次确认退出机制，培养专注习惯

视觉反馈：实时显示剩余时间，专注进度可视化

专注统计：记录每次专注时长，生成学习报告

🤖 AI学习助手
智能答疑：基于OpenAI GPT模型的学习问题解答

解题指导：提供详细的解题步骤和思路分析

多轮对话：支持上下文记忆的连续对话

学习建议：根据学习内容提供个性化学习建议

📚 题目作坊系统
题库管理：树形结构的题目分类管理

闪卡学习：随机抽取题目进行记忆训练

错题收集：自动收集答错的题目，便于复习

学习统计：提供详细的学习进度和掌握情况统计

🃏 快速闪卡功能
随机练习：从题库中随机抽取题目进行练习

即时反馈：点击显示答案，检验学习效果

进度保存：自动保存学习进度，下次继续
